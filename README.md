# 📧 Spam Email Detection using Machine Learning

## 🚀 Project Overview
This project detects whether an email is Spam or Ham using NLP and Machine Learning models.

## 🔧 Models Used
- Logistic Regression
- Multinomial Naive Bayes
- Linear SVM

## 📊 Evaluation Metrics
- Accuracy
- F1 Score
- ROC Curve
- Precision-Recall Curve

## 📈 Features
- TF-IDF Text Vectorization
- Model Comparison
- Visualization Graphs
- Custom Email Prediction
- Streamlit Web App

## 🛠️ Technologies
Python, Scikit-learn, NLP, Matplotlib, Streamlit
